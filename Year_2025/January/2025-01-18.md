## â€‹ My Approach

1. Use three pointers: `prev`, `current`, and `next`.
2. Initialize `prev = NULL`, `current = head`.
3. Iterate while `current` is not NULL:
   - Save `current->next` into `next`.
   - Reverse the link: set `current->next = prev`.
   - Move `prev` to point to `current`.
   - Advance `current` to `next`.
4. Once done, `prev` becomes the new head of the reversed list. Return `prev`.

/* Linked List Node structure:

struct Node
{
    int data;
    struct Node *next;
}

*/

class Solution {
  public:
    Node* reverseList(struct Node* head) {
        // code here
        Node *next = head;
        Node *prev = NULL;
        while(next){
            next = head -> next;
            head -> next = prev;
            prev = head;
            head = next;
        }
        return prev;
    }
};
