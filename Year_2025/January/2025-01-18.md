# ​ Reverse a Linked List

**Problem Link:** [GFG Problem](https://www.geeksforgeeks.org/problems/reverse-a-linked-list/0)

---

### ​ Difficulty

`Easy`

### ​​ Tags

`Linked List`, `Data Structures`

---

## ​ Problem Description

Given the head of a linked list, reverse the list and return the head of the reversed list.

---

## ​ Examples

**Input:**  
head: 1 → 2 → 3 → 4 → NULL

**Output:**  
4 → 3 → 2 → 1

---

**Input:**  
head: 2 → 7 → 10 → 9 → 8 → NULL

**Output:**  
8 → 9 → 10 → 7 → 2

---

**Input:**  
head: 2 → NULL

**Output:**  
2

---

## ​ Constraints

- \(1 \leq \text{list.size}, \text{node->data} \leq 10^5\)

---

## ​ Expected Complexities

- **Time Complexity:** O(n)
- **Auxiliary Space Complexity:** O(1)

---

## ​ My Approach

1. Use three pointers: `prev`, `current`, and `next`.
2. Initialize `prev = NULL`, `current = head`.
3. Iterate while `current` is not NULL:
   - Save `current->next` into `next`.
   - Reverse the link: set `current->next = prev`.
   - Move `prev` to point to `current`.
   - Advance `current` to `next`.
4. Once done, `prev` becomes the new head of the reversed list. Return `prev`.

---

## ​ Code (C++)

```cpp
/* Linked List Node structure:

struct Node
{
    int data;
    struct Node *next;
}

*/

class Solution {
  public:
    Node* reverseList(struct Node* head) {
        // code here
        Node *next = head;
        Node *prev = NULL;
        while(next){
            next = head -> next;
            head -> next = prev;
            prev = head;
            head = next;
        }
        return prev;
    }
};
```
