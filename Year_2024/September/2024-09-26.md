## Problem Statement

You are given the heights of consecutive buildings. You can move from the roof of a building to the roof of the next adjacent building. Find the maximum number of consecutive steps with an increase in altitude with each step.

---

## Examples

**Input:** arr[] = [1, 2, 2, 3, 2]  
**Output:** 1

**Input:** arr[] = [1, 2, 3, 4]  
**Output:** 3

---

## Constraints

- 1 <= arr.size() <= 10^6
- 1 <= arr[i] <= 10^5

---

## Expected Complexities

- Time Complexity: O(n)
- Auxiliary Space: O(1)

---

## Code

```cpp
class Solution {
  public:
    int maxStep(vector<int>& arr) {
        int i = 0, j = 1;
        int c = 0, mx = 0;
        while (j < arr.size()) {
            if (arr[j-1] < arr[j]) {
                c++;
                j++;
            } else {
                mx = max(mx, c);
                c = 0;
                i = j;
                j++;
            }
        }
        mx = max(mx, c);
        return mx;
    }
};
```
