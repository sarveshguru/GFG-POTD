## Problem Statement

Given a queue `q` containing integer elements, reverse the queue.

---

## Examples

**Input:** q[] = [4, 3, 1, 10, 2, 6]  
**Output:** [6, 2, 10, 1, 3, 4]  

**Input:** q[] = [4, 3, 2, 1]  
**Output:** [1, 2, 3, 4]

## Code

```cpp
class Solution {
  public:
    queue<int> rev(queue<int> q) {
        stack<int> s;
        while(!q.empty()){
            s.push(q.front());
            q.pop();
        }
        while(!s.empty()){
            q.push(s.top());
            s.pop();
        }
        return q;
    }
};
```
