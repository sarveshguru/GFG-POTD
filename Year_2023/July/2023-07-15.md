### ğŸš€ My Approach

- Use bit manipulation:
  - Initialize `count = 0`
  - While `N != 0`:
    - If `N & 1` is true, increment `count`
    - Right-shift `N` by 1
- Return `count`

## ğŸ’» Code (C++)

```cpp
class Solution {
  public:
    int setBits(int N) {
        int count = 0;
        while(N != 0){
            if(N & 1)
                count++;
            N >>= 1;
        }
        return count;
    }
};
```
